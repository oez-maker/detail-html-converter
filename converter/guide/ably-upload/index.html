변환한 HTML을 모바일 판매 채널에 적용하기

최종 수정: 2026년 6월 18일 · 예상 읽기 시간 6분

변환 완료는 게시 완료가 아닙니다. 에이블리를 비롯한 판매 채널마다 허용 태그, 이미지 주소, 필수 상품 정보가 다르므로 실제 등록 화면에서 최종 검토해야 합니다.

붙여넣기 전에 사본을 보관하세요

변환 결과를 HTML 파일로 저장하면 채널 편집기에서 코드가 일부 제거되더라도 비교할 수 있습니다. 원본 HTML, 변환 결과, 실제 게시된 결과를 구분해 보관하면 이후 수정도 쉬워집니다.

채널의 입력 방식을 확인하세요

일부 판매 채널은 HTML 직접 입력을 지원하지만, 일부는 이미지와 텍스트 블록만 받습니다. HTML 입력이 지원되지 않는 화면에 코드를 일반 텍스트로 붙여넣으면 태그 자체가 노출될 수 있습니다. 등록 화면의 도움말에서 HTML 모드와 허용 태그를 먼저 확인하세요.

모바일 미리보기에서 볼 항목

  • 가로 스크롤이 생기지 않는지
  • 이미지가 화면보다 넓거나 찌그러지지 않는지
  • 글자 크기와 줄 간격이 읽기 편한지
  • 두 이미지 묶음이 의도한 순서로 표시되는지
  • 외부 이미지가 로그인 없이 로딩되는지
  • 동영상이나 링크가 빈 영역으로 남지 않는지

상품 정보는 별도로 대조하세요

HTML에는 예전 가격, 종료된 행사, 배송 일정 또는 다른 채널명이 남아 있을 수 있습니다. 실제 등록 폼의 상품명, 옵션, 가격, 재고, 배송·교환 안내와 상세페이지 문구가 서로 모순되지 않는지 확인하세요. 식품, 화장품, 어린이 제품, 전기용품 등은 품목별 표시 사항이 있을 수 있습니다.

브랜드와 이미지 권리 확인

원본 상세페이지에 타사 로고, 공급처 문구, 모델 이미지 또는 인증 마크가 있다고 해서 새 채널에서도 사용할 권리가 자동으로 생기지는 않습니다. 변환기는 권리 상태를 판단하지 않으므로 게시자가 사용 범위를 확인해야 합니다.

게시 후 다시 확인하세요

저장 직후 실제 상품 페이지를 모바일과 PC에서 각각 열어봅니다. 편집기 미리보기와 공개 페이지의 CSS가 달라질 수 있기 때문입니다. 이미지가 늦게 뜨거나 문단이 사라지면 채널이 특정 태그나 속성을 제거했는지 결과 HTML과 비교하세요.
