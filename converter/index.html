상세페이지 HTML 변환 방법

원본을 무작정 붙여넣기보다 아래 순서대로 확인하면 이미지 누락이나 깨진 레이아웃을 줄일 수 있습니다.

  1. 원본 HTML 복사스마트스토어 편집기나 보관 중인 상세페이지 소스에서 상품 설명 영역의 HTML을 복사합니다.
  2. 변환 옵션 확인두 사진의 너비와 본문 글씨 크기를 정하고, 판매 채널에서 지원하지 않는 동영상은 제거하도록 설정합니다.
  3. 변환 결과 검토결과 코드와 미리보기를 함께 확인합니다. 이미지 주소, 상품명, 가격 및 필수 고지가 최신인지 직접 점검하세요.
  4. 판매 채널에서 최종 확인결과를 붙여넣은 뒤 모바일 미리보기에서 줄바꿈, 이미지 비율, 금지 태그 여부를 다시 확인합니다.
실제 변환 원리

변환 전후에는 무엇이 달라지나요?

변환 전

<div class="se-component se-text">
  <p class="se-text-paragraph">상품 설명</p>
</div>

편집기 전용 클래스와 중첩 구조가 포함되어 다른 판매 채널에서 모양이 달라질 수 있습니다.

변환 후

<p style="margin:8px 0;">상품 설명</p>

문단과 이미지 중심의 단순한 구조로 바꿔 복사와 수정이 쉬워집니다.

이 도구는 결과를 자동으로 게시하지 않습니다. 각 판매 채널의 최신 작성 정책과 상품별 표시 의무는 판매자가 최종 확인해야 합니다.

변환 중 문제가 생겼을 때

입력한 HTML이 서버에 저장되나요?

아니요. 변환은 현재 브라우저에서 실행되며 이 사이트의 별도 서버로 원본이나 결과를 전송하지 않습니다.

이미지가 미리보기에는 보이는데 판매 채널에서는 깨집니다.

원본 쇼핑몰이 외부 사이트에서의 이미지 호출을 막았을 가능성이 큽니다. 이미지를 내려받아 사용 권한을 확인한 뒤 판매 채널의 이미지 업로드 기능을 이용하세요.

모든 스마트스토어 HTML을 변환할 수 있나요?

스마트에디터 계열의 문단과 이미지 구조를 중심으로 지원합니다. 맞춤 스크립트, 표, 슬라이드, 지도와 같은 복잡한 구성은 일부 누락될 수 있습니다.

동영상을 유지해도 되나요?

판매 채널이 iframe이나 외부 동영상을 허용하는지 먼저 확인해야 합니다. 확실하지 않다면 동영상 제거 옵션을 사용하는 편이 안전합니다.

결과 HTML이 너무 길면 어떻게 하나요?

중복 설명과 불필요한 이미지를 줄이고, 이미지 파일을 판매 채널에 직접 업로드한 뒤 생성된 주소로 교체하세요.

변환 후에도 글자 모양이 달라집니다.

판매 채널의 기본 CSS가 적용되기 때문입니다. 본문 글씨 크기를 조절하되 특정 글꼴에 의존하지 않는 구성이 안정적입니다.

결과를 바로 상품 페이지에 게시해도 되나요?

미리보기는 참고용입니다. 상품 정보, 저작권, 표시 의무와 채널 정책을 직접 검토한 뒤 게시해야 합니다.

오류를 제보하려면 무엇을 보내야 하나요?

브라우저 종류, 오류 메시지, 개인정보를 제거한 최소한의 재현용 HTML을 문의 이메일로 보내주세요.

더 자세한 설명이 필요하신가요?

원본 준비부터 이미지 오류 해결까지 주제별 가이드에서 단계별로 확인할 수 있습니다.
