상세페이지 HTML 변환기 소개

상세페이지 HTML 변환기는 쇼핑몰 운영자가 스마트스토어 등에서 작성한 상세페이지 HTML을 에이블리 등 다른 판매 채널에 맞게 빠르게 정리할 수 있도록 만든 브라우저 기반 도구입니다.

현재 도구는 HTML 안의 텍스트, 이미지, 일부 레이아웃 정보를 읽어 중앙 정렬 상세페이지 형태로 재구성합니다. 동영상, 외부 임베드, 불필요한 편집기 전용 클래스는 옵션에 따라 제거됩니다.

처리 방식

입력한 HTML은 사용자의 브라우저 안에서만 변환됩니다. 이 사이트는 변환할 HTML을 별도 서버로 업로드하거나 저장하지 않습니다.

주의 사항

⚠️ 안전한 거래 및 플랫폼 정책 준수

외부 이미지 주소, 타사 플랫폼 문구, 브랜드 로고, 저작권이 있는 설명문은 판매 채널 정책에 따라 문제가 될 수 있습니다. 변환 결과를 게시하기 전 운영자가 직접 확인해야 합니다.
