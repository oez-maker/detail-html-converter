상세페이지 HTML 변환기 소개

상세페이지 HTML 변환기는 쇼핑몰 운영자가 스마트스토어 등에서 작성한 상세페이지 HTML을 에이블리 등 다른 판매 채널에 맞게 빠르게 정리할 수 있도록 만든 브라우저 기반 도구입니다.

현재 도구는 HTML 안의 텍스트, 이미지, 일부 레이아웃 정보를 읽어 중앙 정렬 상세페이지 형태로 재구성합니다. 동영상, 외부 임베드, 불필요한 편집기 전용 클래스는 옵션에 따라 제거됩니다.

처리 방식

입력한 HTML은 사용자의 브라우저 안에서만 변환됩니다. 이 사이트는 변환할 HTML을 별도 서버로 업로드하거나 저장하지 않습니다.

만든 이유

판매 채널마다 허용하는 HTML과 이미지 처리 방식이 달라 같은 상세페이지를 옮길 때 반복적인 정리 작업이 생깁니다. 이 도구는 그중 문단, 이미지, 캡션처럼 공통으로 자주 쓰이는 요소를 읽기 쉬운 코드로 바꾸는 데 초점을 맞췄습니다.

완전한 자동 게시 도구가 아니라 운영자가 결과를 검토하기 쉽게 만드는 보조 도구입니다. 변환 결과는 미리보기와 코드로 함께 제공되며, 최종 게시 여부와 상품 정보의 정확성은 사용자가 판단합니다.

운영 원칙

  • 변환에 입력한 HTML을 별도 서버에 저장하지 않습니다.
  • 지원하지 않는 요소와 알려진 제한 사항을 가이드에 공개합니다.
  • 특정 판매 채널과의 공식 제휴 또는 보증을 주장하지 않습니다.
  • 오류 제보를 바탕으로 변환 결과와 설명을 지속적으로 개선합니다.

주의 사항

⚠️ 안전한 거래 및 플랫폼 정책 준수

외부 이미지 주소, 타사 플랫폼 문구, 브랜드 로고, 저작권이 있는 설명문은 판매 채널 정책에 따라 문제가 될 수 있습니다. 변환 결과를 게시하기 전 운영자가 직접 확인해야 합니다.
